    使用说明

    键盘弹奏

    开启顶栏 ⌨ 键盘弹奏 后，在非输入框焦点下按键即可演奏（底部钢琴同步发光）。在简谱框内输入时仍正常打字；按住 Alt 可在输入框内弹奏。

    按键音简谱
    1–7 或 A–J中音 do–si1–7
    Z–M低音(1)–(7)
    Q–U高音[1]–[7]
    Shift+数字 / Shift+Q~U升半音#1 等

    也可直接用鼠标点击底部钢琴键。点击顶栏「键盘弹奏」可开关。

    基本规则

    本软件采用 C 大调数字简谱：用 1–7 表示 do–si，不记录时值强弱，通过休止控制节奏。

    写法含义
    1 – 7中音 do – si
    #3升半音（如 #4、#5）
    (5)低一个八度
    ((5))低两个八度
    [1]高一个八度
    [[1]]高两个八度
    - 空格 换行休止一拍（时长由速度滑块决定）

    示例

    1155665 4433221 — 小星星
    1231 1231 345 345 — 两只老虎

    快捷键

    Ctrl + Enter 播放
    Esc 停止
    空格（非输入框）播放 / 停止

    线稿转数字谱

    切换到「线稿 → 数字谱」，在网格上画出旋律曲线，点转为数字简谱自动写入编辑器。

    曲库

    左侧可导出曲库 / 导入曲库，+ 收录 保存当前曲目。

    数据

    顶栏「导出」为当前单首简谱；编辑内容会自动保存在本机。
